# retail-sales-data-pipeline-azure
# Retail Sales Data Pipeline using Azure

## Overview
This project demonstrates an end-to-end ETL pipeline using Azure services.

## Project Structure
- data/ : Contains raw dataset
- notebooks/ : PySpark transformation scripts
- pipeline/ : ADF pipeline JSON
  
## Tools Used
- Azure Data Factory (ADF)
- Azure Data Lake Storage (ADLS Gen2)
- Azure Databricks
- PySpark

## Workflow
1. Data ingestion using Azure Data Factory
2. Data stored in Azure Data Lake
3. Data transformation using PySpark in Databricks
4. Aggregated data used for analytics

## Features
- Data cleaning and filtering
- Aggregation by city
- Simulated ETL pipeline

## Note
This project is created based on hands-on practice with Azure services using free credits.
